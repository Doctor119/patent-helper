A Message for Graders

Thank you for taking the time to grade my capstone project. I hope the navigation through the pages will be self-explanatory, but if not, you can see the app's functionality demonstrated by doing the following:

  1. Put yourself in the shoes of a small-time inventor and think of a quick description for an invention you might want to patent, 2-3 sentences.
  2. Go to the Find Your CPC page and use the search functions there to find the CPC of your invention.
  3. Go to the Preliminary Search page. Enter your invention description and CPC into the appropriate search fields.
  4. Look at your search results from the patent database. You can also remove the CPC code and search again for different (probably less useful) results.

See the About page for details about myself and this app.
